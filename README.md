# JS-Games
JS Canvas Gaming Examples

## Flight
Flight Simulator

## Moonlander
Lunar Lander Var

## Platformer
2d Platformer

# Run
No assembly required!  Just open `index.html` in each games folder.  Use Chrome Developer Tools to debug while running.

# Build

## Container
Build container using command:

`docker build js_games .`

Runs on port 8080
